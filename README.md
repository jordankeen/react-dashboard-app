

# React Dashboard App

Static dashboard app using,

- React.js
- Material UI
- React Pro Sidebar
- formik (form)
- yup (form validation)
- Datagrid from Material UI
- Full Calendar
- Nivo Charts

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/jordankeen/react-dashboard-app.git

2. npm install

3. npm start
